# Soil CNN Model for SMART Agriculture 🌱🤖

A Deep Learning based Soil Classification and Crop Recommendation System using Convolutional Neural Networks (CNN).

This project helps identify soil types from images and provides smart agricultural recommendations such as suitable crops, water requirements, and soil pH ranges.

---

## 🚀 Live Demo

🌐 Deployed Project: https://smart-agriculture-y8c4.onrender.com/

---

## 🚀 Features

- Soil image classification using CNN
- Predicts soil type from uploaded image
- Crop recommendation system
- Water requirement suggestion
- Soil pH range recommendation
- Deep Learning model trained using TensorFlow/Keras
- Easy deployment using Flask/Streamlit/Gradio

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/prince12raj/soil-cnn-model.git
cd soil-cnn-model
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## ▶️ Training the Model

```bash
python train.py
```

---

## 🔍 Predict Soil Type

```bash
python predict.py
```

Run the web application:

```bash
python app.py
```
